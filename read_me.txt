Формат ввода и функции БД:

Дата вводится в формате гггг-мм-дд, дата - date
Событие вводится на английском языкеБ событие - event


Add date event = добавить событие в конкретный день.
Find date = распечатать события за конкретный день.
Del date event = удалить конкретное событие в конкретный день.
Del date = удалить все собтия за конкретный день.
Print = печать всех событий.
